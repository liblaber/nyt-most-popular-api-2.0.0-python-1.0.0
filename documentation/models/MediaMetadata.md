# MediaMetadata

**Properties**

| Name   | Type | Required | Description                |
| :----- | :--- | :------- | :------------------------- |
| url    | str  | ❌       | Image's URL.               |
| format | str  | ❌       | Image's crop name.         |
| height | int  | ❌       | Image's height (e.g. 293). |
| width  | int  | ❌       | Image's width (e.g. 440).  |
